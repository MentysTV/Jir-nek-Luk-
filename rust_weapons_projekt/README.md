# ⚔️ Rust Weapons - Python List Operations

Projekt demonstruje základní operace se seznamem (list) v Pythonu na příkladu zbraní z hry Rust.

## 📝 Zadání

Program provádí následující operace:

1. **Vytvoření listu** ze zkopírovaného seznamu zbraní z hry Rust
2. **Přidání položky** do listu
3. **Odstranění první položky** z listu
4. **Vypsání délky** listu
5. **Vypsání celého listu** s číslováním

## 🚀 Spuštění programu

```bash
python rust_weapons.py
```

## 📊 Výstup programu

Program vypíše:
- Původní stav seznamu (44 zbraní)
- Stav po přidání položky (Snowball Gun)
- Stav po odstranění první položky (Revolver)
- Finální seznam všech zbraní s číslováním
