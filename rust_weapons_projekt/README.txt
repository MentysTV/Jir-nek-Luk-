⚔️ PROJEKT: PRÁCE SE SEZNAMEM ZBRANÍ V RUSTU
===============================================

📁 OBSAH SLOŽKY:
- rust_weapons.py - hlavní Python program
- README.txt - tento soubor s popisem

📝 POPIS PROGRAMU:
Program demonstruje základní operace se seznamem (list) v Pythonu:

1. Vytvoření listu ze zkopírovaného seznamu zbraní z hry Rust (44 položek)
2. Přidání další položky do listu (Snowball Gun)
3. Odstranění první položky z listu (Revolver)
4. Vypsání délky listu
5. Vypsání celého listu s číslováním

🚀 SPUŠTĚNÍ:
1. Otevřete příkazový řádek nebo PowerShell
2. Přejděte do složky: cd C:\rust_weapons_projekt
3. Spusťte program: python rust_weapons.py

📊 VÝSTUP:
Program vypíše:
- Původní stav seznamu
- Stav po přidání položky
- Stav po odstranění první položky
- Finální seznam všech zbraní (číslovaný)

💻 POŽADAVKY:
- Python 3.x
